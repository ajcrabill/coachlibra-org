# coachlibra.org — Coach Libra Legacy

Marketing site for **Coach Libra Legacy** — family legacy books. A parent's life stories, captured in their own voice, woven with their photographs, and kept forever. Built for adult children creating a keepsake for an aging parent.

A sibling brand of [coachlibra.com](https://coachlibra.com) (author coaching for founders & leaders), sharing the same engine and brand family (Fraunces + Hanken Grotesk, warm cream / oxblood / brass).

## Stack
Static site, hosted on GitHub Pages at the apex domain `coachlibra.org` (see `CNAME`). No build step — plain HTML/CSS/JS in `index.html`. `.nojekyll` keeps Pages from processing the files.
